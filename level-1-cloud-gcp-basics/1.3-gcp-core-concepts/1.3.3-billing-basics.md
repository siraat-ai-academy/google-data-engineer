- Compute usage (CPU, memory)

- Data processing
- Network usage

## 7. Pay-as-You-Go (Very Important)

Pay-as-you-go means:
- No upfront payment
- No long-term contract
- Pay only for usage

Example:
- Use server for 1 hour â†’ pay for 1 hour
- Stop server â†’ billing stops

This gives:
- Cost flexibility
- Low risk for beginners
- Easy experimentation

## 8. Budgets and Alerts (Cost Control)

GCP allows you to:
- Set **budgets**
- Create **alerts**

Example:
- Budget: $50 per month
- Alert at: 80%, 90%, 100%

This helps:
- Avoid surprise bills
- Control learning cost
- Protect company budget
