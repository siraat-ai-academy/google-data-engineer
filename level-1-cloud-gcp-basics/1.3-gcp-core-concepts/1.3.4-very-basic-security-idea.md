## 3. Shared Responsibility Model (Very Important)

Cloud security is **shared** between:

### Google (Cloud Provider)
Google is responsible for:
- Data centers
- Physical security
- Network infrastructure
- Hardware protection

### You (Customer)

- Who has access

- They cannot see anything inside it

- Projects are powerful
- Access must be controlled carefully
