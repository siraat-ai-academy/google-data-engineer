# 0.2.2 Difference between Data Engineer and Data Analyst

---

## 1. Concept Explanation 

Both roles work with data, but **their purpose is very different**.

- **Data Engineer**:
  > Builds systems that prepare and deliver data

- **Data Analyst**:
  > Uses prepared data to find answers and insights

Think like this:

> **Data Engineer = builds the kitchen**  
> **Data Analyst = cooks food in that kitchen**

---

## 2. Main Responsibility Difference

### Data Engineer:
- Collects raw data
- Cleans and transforms data
- Builds data pipelines
- Stores data safely
- Ensures data arrives on time

### Data Analyst:
- Uses cleaned data
- Writes queries for analysis
- Creates reports and dashboards
- Answers business questions
- Explains results to stakeholders

## 6. Real-World Example (Very Clear)

### Example: Online Shopping Company

#### Data Engineer:
- Builds pipeline to collect orders
- Cleans duplicate records
- Stores data in warehouse
- Makes data available daily

#### Data Analyst:
- Uses that data
- Finds daily sales
- Identifies best products
- Shares insights with managers

Without Data Engineer:
- Analyst cannot work

Without Data Analyst:
- Business cannot understand data
