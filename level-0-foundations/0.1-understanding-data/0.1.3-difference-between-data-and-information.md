# 0.1.3 Difference between data and information

---

## 1. Concept Explanation (Very Simple English)

The difference is simple:

- **Data** = raw facts (no meaning by itself)
- **Information** = data that is processed and understood

Think like this:

> **Data answers ‚Äúwhat?‚Äù**  
> **Information answers ‚Äúwhat does it mean?‚Äù**

## 2. Simple One-Line Definition

- **Data:** Raw values without context  
- **Information:** Meaningful result after processing data

## 6. Why Companies Care About This Difference

Companies do NOT make decisions on raw data.

They:
1. Collect data
2. Clean data
3. Process data
4. Create information
5. Take decisions

If data is wrong:
- Information becomes wrong
- Decisions fail
- Business suffers

That is why **data engineering exists**.
