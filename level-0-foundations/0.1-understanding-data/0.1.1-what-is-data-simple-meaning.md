# 0.1.1 What is data (simple meaning)

---

## 1. Concept Explanation 

**Data** means:

> **Raw facts or values that are collected, but not yet explained.**

Data by itself:
- Has no meaning
- Is not a decision
- Is just a record

Think of data as:
- Numbers
- Text
- Dates
- Signals

When data is **processed and understood**, it becomes **information**.

## 3. Why Data Alone Is Not Enough

Data is:
- Unorganised
- Unclean
- Sometimes incorrect

Companies do NOT directly use raw data.

They first:
- Collect data
- Clean data
- Transform data
- Analyze data

Only then:
- Decisions are made
- Reports are created
- Business actions happen

---

## 4. Why Companies Collect Data

Companies collect data to:

- Understand customers
- Improve products
- Reduce cost
- Increase profit
- Predict future trends

Examples:
- E-commerce tracks clicks
- Banks track transactions
- Hospitals track patient records
- Apps track user behaviour

All of this starts with **data**.
