# 0.3.1 What is a computer system

---

## 1. Concept Explanation 

A **computer system** is:

> **A machine that takes input, processes it, and gives output.**

In simple words:
- You give data to the computer
- Computer works on it
- Computer gives result

This process happens **very fast** and **accurately**.

---

## 2. Simple Formula (Very Important)

A computer system always works like this:

**Input ‚Üí Process ‚Üí Output**

Example:
- Input: Numbers you type
- Process: Calculation
- Output: Answer on screen

---

## 3. Real-Life Examples of a Computer System

### Example 1: Calculator
- Input: `5 + 3`
- Process: Addition
- Output: `8`

---

### Example 2: Mobile Phone
- Input: Touch, voice, photos
- Process: Apps + CPU
- Output: Screen display, sound

---

### Example 3: ATM Machine
- Input: Card + PIN
- Process: Bank system checks data
- Output: Cash / balance slip

All are **computer systems**.
